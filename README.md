# InfoAquarium

> Динамический граф как информационный аквариум — путешествие по смыслу через стеклянный муравейник.

Граф одновременно является **картой мира** и **движком графики**: узлы — понятия/заметки, рёбра — связи и пути путешественника, треугольники Делоне между узлами — цветной ландшафт (горы, долины, пещеры). Пространство — живое: физическая симуляция двигает узлы каждый кадр.

---

## Структура

```
1-browser/        Автономный прототип — один HTML-файл, работает без сборки
2-react/          React + Vite + react-force-graph (3D) + Delaunay terrain
3-obsidian/       Плагин Obsidian — граф из твоих заметок как живой аквариум
```

---

## 1. Browser Prototype

Открыть напрямую в браузере:

```bash
open 1-browser/index.html
# или
python3 -m http.server 8080 --directory 1-browser
```

**Управление:**
- Клик на узел — путешественник строит BFS-маршрут до цели
- Перетащить фон — панорама
- Колесо мыши — зум

**Что внутри:**
- Физика: отталкивание (Кулон) + пружины (Гук) + демпфирование
- Триангуляция Делоне — треугольники между узлами, раскрашенные по семантической категории
- Путешественник (`traveler`) — светящаяся точка, движется вдоль рёбер, оставляет след
- Поиск пути: BFS по графу

## 2. React App

```bash
cd 2-react
npm install
npm run dev
```

**Стек:** React 18 · Vite · `react-force-graph` (Three.js WebGL) · `delaunator`

**Возможности:**
- 3D-граф с физикой (`d3-force-3d`)
- Прозрачные узлы-сферы с wireframe-оболочкой
- Terrain overlay: треугольники Делоне поверх 3D как 2D canvas
- Временной ползунок — фильтрует граф по эпохам (какие группы активны)
- Путешественник: движется в 3D-пространстве, камера летит к целевому узлу
- BFS по видимым рёбрам текущей эпохи

## 3. Obsidian Plugin

```bash
cd 3-obsidian
npm install
npm run build
```

Скопировать `main.js` + `manifest.json` в папку плагина Obsidian:
```
<vault>/.obsidian/plugins/info-aquarium/
```

**Команда:** `InfoAquarium: Open InfoAquarium graph`
**Иконка:** `git-fork` в левой панели

**Что делает:**
- Читает все заметки vault через `MetadataCache`
- Строит граф: заметки = узлы, внутренние ссылки `[[...]]` = рёбра
- Размер узла пропорционален числу ссылок
- Живая физика, триангуляция Делоне, путешественник
- Клик на узел — маршрут и фокус камеры
- Если vault пуст — показывает демо-граф

## Концепция

| Метафора | Техника |
|----------|---------|
| Стеклянный муравейник | Прозрачные узлы + terrain triangles |
| Рёбра = дороги мира | Путешественник движется только по рёбрам |
| Горы мира = полигоны | Delaunay triangulation + цвет по категории |
| Мир внутрь, не наружу | Hyperbolic tree (d3-hypertree) — следующий шаг |
| Время как измерение | Ползунок эпох, фильтрация рёбер |
| Информационный запах | ACO pheromone trails (планируется) |
